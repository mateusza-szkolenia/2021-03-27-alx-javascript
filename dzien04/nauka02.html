ścieżka w windows: 

    / slash         (ukośnik)
        - dzielenie
        - w adresach URL - oddziela katalogi
        - w ścieżkach w systemach operacyjnych Unix, Linux, macOS
        - zamykanie tagów np. </html>
        - komentarze blokowe w JS /* komentarz */
        - komentarze do końca linii w JS // komentarz

    \ backslash     (ukośnik odwrotny)
        - w ścieżkach w systemie Windows
        - w językach programowania do tzw. eskejpowania (escape)
