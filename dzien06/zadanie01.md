# Zadanie

Stworzyć obiekt Prostokat, który zawiera w sobie w własności: `a` i `b` (`wysokosc`, `szerokosc`) oraz 3 metody:
- `.pole()`
- `.obwod()`
- `.przekatna()`

## Przykład użycia
```javascript

Prostokat.a = 30
Prostokat.b = 40
Prostokat.pole() // zwraca 1200
Prostokat.obwod() // zwraca 140
Prostokat.przekatna() // zwraca 50
```
